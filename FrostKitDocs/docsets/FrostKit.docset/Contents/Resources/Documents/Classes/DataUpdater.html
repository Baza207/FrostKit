DataUpdater

The data updater is a class that helpers update data from a FUS based system. It is made to be inserted into IB via an NSObject.It handles controlling loading, updating, saving of data along wit using the request store and data stores to mange these functions. It is also written to work with a data soruce for either UITableView or UICollectionView.
